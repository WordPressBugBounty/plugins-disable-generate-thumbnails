== Description ==

= Thumbnails =
* thumbnail
* medium
* large
* 1536x1536
* 2048x2048
* Generated with theme
* Generated with plugin

= Functions =
* Large image threshold
* Exif automatic rotation

== Installation ==

1. Upload `disable-generate-thumbnails` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
